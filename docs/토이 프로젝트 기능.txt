index.html
 db에서 불러온 지도의 목록을 볼 수 있다
 검색버튼을 통해 db에 조건에 따라 필터링된 지도 목록을 요청할 수 있다
  -구글 geocoding api를 통해 문자를 좌표로 변환해 서버에서 $near연산을 통해 반환받는다
 지도의 목록을 클릭해 지도 상세보기 페이지로 이동할 수 있다
  -파라미터는 map_number
 회원 아이콘을 클릭해 로그인할 수 있다
  -로그인하면 회원 아이콘이 회원정보 수정, 로그아웃 버튼으로 변경된다
 홈 아이콘을 클릭해 index.html로 돌아올 수 있다
 
지도상세.html
 index.html에서 받아온 map_number에 해당하는 지도를 db에 요청하여 보여줄 수 있다
  -db에는 지도의 인덱스와 지도에 찍힌 마커들의 정보만 저장되며 클라이언트의 브라우저에서 구글 api에 렌더링된다
 follow 버튼을 누르면 현제 세션에 저장된 유저의 정보에 해당 지도의 map_number를 추가해준다
 뒤로가기, 돌아가기 버튼을 누르면 index.html로 돌아갈 수 있다(혹은 새창으로 열어도 무방)
 
나만의지도.html
 현제 세션에 저장된 유저 정보에서 유저가 제작한 지도 번호들의 목록을 좌측에 보여준다
 우측에는 유저의 지도 번호에 맞는 지도 정보를 보여준다
  -지도에는 마커를 추가하거나 삭제하거나 이동시키는 동작을 할 수 있다
 좌측의 지도 목록을 클릭하면 우측의 지도가 변경된다
 우하단의 공유 트리거를 누르면 공유상태를 변경할 수있다
 우하단의 저장 버튼을 누르면 db에 저장된 지도를 현재 상태로 업데이트한다

팔로우한 지도.html
  현제 세션에 저장된 유저 정보에서 유저가 팔로우한 지도 번호들의 목록을 좌측 사이드바에 보여준다
  우측에는 유저의 지도 번호에 맞는 지도 정보를 보여준다
